Supported Languages

English Korean Japanese Chinese (Simplified)

Language Directions

LanguageTranslation Available To
KoreanEnglish, Japanese, Chinese (Simplified)
EnglishKorean, Japanese
JapaneseKorean, English
Chinese (Simplified)Korean

Reverse directions are supported, e.g., Korean → English and Japanese → Korean.
Chinese (Simplified) is reached via routed chain English → Korean → Chinese.

Supported File Formats

  • PPTX — PowerPoint Slides
  • DOCX — Word Documents
  • PDF

Maximum file size: 100 MB.

How to Translate

  1. Open DeckLingo AI.
  2. Upload your file (PPTX, DOCX, or PDF).
  3. Select Source Language — the language your original PPT/document is in.
  4. Select Target Language.
  5. Click Translate.
  6. Your translated file will download automatically.
Please use it carefully: we are charged for tokens per translation.
